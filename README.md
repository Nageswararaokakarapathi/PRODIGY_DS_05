# 🚦 Traffic Accident Data Analysis

> **Prodigy InfoTech – Data Science Internship | Task 05**

## 📌 Project Overview

This project was completed as part of the **Prodigy InfoTech Data Science Internship**.

The objective is to analyze real-world traffic accident data and identify patterns related to:

- 🌦️ Weather Conditions
- 🛣️ Road Surface Conditions
- 💡 Light Conditions
- ⏰ Time of Day
- 🚗 Speed Limits
- ⚠️ Accident Severity

Using Exploratory Data Analysis (EDA), various visualizations were created to uncover meaningful insights that can support road safety improvements.

---

## 🎯 Objective

Analyze traffic accident data to identify patterns related to road conditions, weather, lighting, and time of day. Visualize accident hotspots and contributing factors using Python.

---

## 📂 Dataset

**Dataset:** UK Traffic Accidents (2016)

The dataset contains information about road accidents, including:

- Accident Severity
- Weather Conditions
- Road Surface Conditions
- Light Conditions
- Speed Limit
- Date & Time
- Number of Vehicles
- Number of Casualties

## 💡 Key Insights

- Most accidents occurred during daylight hours.
- Fine weather recorded the highest number of accidents.
- Urban roads experienced more accidents than rural roads.
- Peak traffic hours showed higher accident frequency.
- Road surface conditions influenced accident severity.
- Speed limits showed an impact on accident outcomes.
